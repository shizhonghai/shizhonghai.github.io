JavaScript 只出现一次的数字

发表于 2020-10-27 | 阅读次数 | 本文总阅读量 次

题目描述

给定一个非空整数数组，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

说明：

你的算法应该具有线性时间复杂度。 你可以不使用额外空间来实现吗？

示例 1:

输入: [2,2,1]
输出: 1

示例 2:

输入: [4,1,2,1,2]
输出: 4

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/single-number

代码

离散数学的异或运算规则

  • 交换律：a ^ b ^ c => a ^ c ^ b
  • 任何数于0异或为任何数 0 ^ n => n
  • 相同的数异或为0: n ^ n => 0

解题思路

利用异或运算可以找出数组中奇个数的项
示例：[4,1,2,1,2]
解析：4 ^ 1 ^ 2 ^ 1 ^2 等价于 1 ^ 1 ^ 2 ^ 2 ^ 4 => 0 ^ 0 ^ 4 => 4
